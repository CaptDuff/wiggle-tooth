# Wiggle Tooth 🦷

The official reporting portal for the **Tooth Fairy Guild** — where parents file a
report about a child's wiggly (and soon-to-be-lost) tooth so the fairies can plan
their nightly routes.

A single, self-contained static site (`index.html`, no build step, no backend).

## Hosting

Deployed with **GitHub Pages** at [wiggle-tooth.com](https://wiggle-tooth.com).
DNS is managed on Cloudflare. The `CNAME` file sets the custom domain.

## Editing

Everything lives in `index.html` (HTML, CSS, and JS inline). Edit it, commit, and
push to `main` — GitHub Pages redeploys automatically within a minute or two.
